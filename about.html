About The Project

A practical website for helping people choose the right smartphone.

PhoneDSS is a decision support system designed for users who want to compare phones, evaluate alternatives, and receive a recommendation based on their personal needs.

Project Focus Comparison + Evaluation + Recommendation

This is not an e-commerce website. It is a guided decision support website.

Purpose

Help users make better smartphone choices using structured criteria and weighted scoring.

Scope

Focus on iPhone and Samsung models from 2010 to 2025 using image-based product cards and dataset analytics.

Outcome

Return a ranking list, winner result, recommendation explanation, and supportive analytics.
