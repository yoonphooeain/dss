About The Project

A decision support website designed to make smartphone selection clearer, smarter, and more explainable.

PhoneDSS is a web-based Decision Support System created to help users compare smartphones, assign personal preferences, and receive a structured recommendation instead of relying only on brand popularity or informal opinion.

Decision Support System Multi-Criteria Evaluation iPhone vs Samsung
Project Focus Comparison + Weighting + Recommendation

This is not an e-commerce store. It is a guided DSS platform built for evaluation, ranking, and explanation.

Background

Choosing a smartphone is a multi-criteria problem because users usually consider price, performance, camera quality, battery life, display experience, and software support at the same time.

Problem

Many buyers make decisions using scattered reviews, marketing claims, or brand preference alone, which can lead to inconsistent and less transparent choices.

Purpose

The purpose of this project is to provide a structured and explainable framework that helps users identify the most suitable phone under their own priorities.

Research Objective

What the system is designed to achieve

The main objective of PhoneDSS is to transform smartphone comparison into a clearer decision process by applying a weighted scoring model to multiple alternatives. Instead of only showing specifications, the system evaluates phones according to user-defined importance levels and produces a ranked result.

Primary objective Support rational phone selection
Decision model Weighted multi-criteria scoring
Main output Ranking + winner + explanation
How It Works

From user input to final recommendation

The system follows a simple but structured flow. Users first browse the smartphone dataset, then move to the DSS evaluation page where they select phones and assign weights to the six criteria. The system calculates weighted scores, ranks the alternatives, identifies the highest-scoring phone, and provides explanation and analytics to support the result.

Step 1 Select phones from the dataset
Step 2 Assign weights to six criteria
Step 3 Calculate weighted scores
Step 4 Review ranking, winner, and charts
Scope of the Study

Current dataset and boundaries

The current scope of PhoneDSS focuses on Apple iPhone and Samsung Galaxy smartphone models from 2010 to 2025. These two brands were selected because they are widely recognized, strongly represented in the premium smartphone market, and often compared directly by users. The system evaluates models using six criteria: price, performance, camera, battery, display, and software support.

The website is designed as a decision support platform rather than a live commercial marketplace. Its role is to compare, evaluate, and recommend phones based on structured criteria and user preference weighting.

Key Features

1 112 smartphone models
2 6 weighted DSS criteria
3 Live ranking and score calculation
4 Winner explanation output

Research Significance

PhoneDSS demonstrates how a web-based Multi-Criteria Decision Making approach can be applied to a real consumer problem and translated into an explainable, user-centered recommendation system.

Outcome

The system returns a ranking list, identifies the winning phone, explains why it wins, and supports the recommendation with comparative analytics.

Value for Thesis Work

This project shows the practical use of Decision Support Systems, weighted scoring logic, and user-centered evaluation in a real-world technology selection context.

Expected Benefit

Users can make more transparent and personalized smartphone decisions, while the project itself serves as a research-oriented example of explainable digital decision support.
